This plugin adds a coupon checkbox to WooCommerce products.

== Description ==

The RCKFLR Coupon Checkbox plugin allows you to add a coupon checkbox to your WooCommerce products. When a customer visits a product page that has a coupon associated with it, they will see a checkbox that allows them to apply the coupon to their purchase.

== Installation ==

1. Upload the `rckflr-coupon-checkbox` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the edit page for a WooCommerce product and enter a coupon code in the 'Coupon Code' metabox
4. Visit the product page on the front end of your site. If the coupon is valid, you will see a checkbox that allows you to apply the coupon.

== Frequently Asked Questions ==

= Does this plugin work with all themes? =

This plugin should work with most themes, but it may not work with themes that significantly alter the default WooCommerce product page layout.

= Can I use this plugin with other WooCommerce plugins? =

Yes, this plugin should be compatible with most other WooCommerce plugins.
